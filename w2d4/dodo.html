CSS SELECTORS

CSS Selectors use to find the elements inside your HTML file. With the CSS Selectors, you can easily choose and change the style you want.

TRY IT!
SELECTOR EXAMPLE DESCRIPTION
*{} *{} Selects all elements
# #main{} Selects the elements by their id's
. .main{} Selects the elements by their class's
a, p, h1,... a{}, p{}, h1{},... Selects all elements related to the tags you wrote
elements.class p.main{}, a.main{},... Selects only p/a elements with clas=main
elements,elements,... div{},a{},p{},... Selects all div, a and p elements

CSS ANIMATIONS

CSS Animations use to create animations inside for your webpage without using javascript or flash!

In order to animate the element that you select, you need to use @keyframe rule.

HOVER
#test:hover{ animation-name: move; animation-duration: 6s; }
@keyframes move { from {background-color: #f8edeb; transform: translateX(100px);} to {background-color: aquamarine; transform: translateX(500px;)} }
